## 名字

-   芸轶
    -   书籍。芸，香草，置书页内可以辟蠹。

-   刈楚
    -   翘翘错薪，言刈其楚。
    -   出处：《诗经，汉广》
-   隼庭
    -   仲尼在陈﹐有隼集于陈侯之庭而死﹐楛矢贯之﹐石砮其长尺有咫。陈惠公使人以隼如仲尼之馆问之。仲尼曰﹕'隼之来也远矣﹐此肃慎氏之矢也……君若使有司求诸故府﹐其可得也。'使求得之金椟﹐如之。
    -   出处：《国语．鲁语下》
-   秉彝
    -   天生烝民，有物有则。民之秉彝，好是懿德。
    -   出处：《诗·大雅·烝民》
